# 📚 LIB – Accessible Digital Library Website

## 🌐 Overview

**LIB** is a responsive digital library website designed to help users browse, search, and explore books and learning resources in a clear, structured, and accessible way.

The project focuses on simplicity, readability, and inclusive design rather than building a full-scale library system. It supports users with different technical backgrounds and works across desktop and mobile devices.

---

### Run Locally

- Open `index.html` in your browser  
- No installation or build tools required (pure HTML, CSS, JS)

## ✨ Features

- 📄 Multi-page website (Home, About, Catalog, Resources, Contact)
- 🔍 Search & Filter Catalog
- ⚡ Dynamic content rendering with JavaScript
- 🧭 Navigation with header and breadcrumbs
- 🪟 Accessible modal dialog for book details
- 📝 Contact form with validation
- 📱 Responsive design (Grid + Flexbox)
- 🎨 Consistent design system

---

## ♿ Accessibility

- Semantic HTML structure  
- ARIA attributes for assistive technologies  
- Keyboard navigation with visible focus states  
- Accessible forms with labels and error messaging  
- Responsive layout for all devices  
- High contrast and readable typography  

---

## ⚠️ Known Limitations

- No backend (form does not send data)
- Static dataset (no database integration)
- Limited scalability for large catalogs
- Basic search/filter logic
- Dialog accessibility can be improved

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Grid + Flexbox)  
- JavaScript (Vanilla JS)
